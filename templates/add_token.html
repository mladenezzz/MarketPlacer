Где найти токены:

Wildberries:

  1. Войдите в личный кабинет продавца
  2. Перейдите в Настройки → Доступ к API
  3. Создайте новый токен с нужными правами
  4. Скопируйте токен (он показывается только один раз!)

Ozon:

  1. Войдите в личный кабинет продавца
  2. Перейдите в Настройки → Seller API
  3. Скопируйте Client ID
  4. Создайте API-ключ и скопируйте его
