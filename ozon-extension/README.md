# OZON Product Info Extension

Chrome-расширение для отображения информации о товарах из базы MarketPlacer при наведении на артикул в ЛК продавца OZON.

## Установка

1. Откройте Chrome и перейдите в `chrome://extensions/`
2. Включите "Режим разработчика" (Developer mode) в правом верхнем углу
3. Нажмите "Загрузить распакованное расширение" (Load unpacked)
4. Выберите папку `ozon-extension`

## Настройка

### Иконки
Добавьте файлы иконок в папку расширения:
- `icon16.png` (16x16 px)
- `icon48.png` (48x48 px)
- `icon128.png` (128x128 px)

Можно использовать любое изображение или сгенерировать простую иконку.

### API сервер
По умолчанию расширение обращается к `http://192.168.0.44:5000/api/extension`.
Если адрес сервера другой, измените `CONFIG.API_BASE` в файле `content.js`.

## Использование

1. Перейдите на https://seller.ozon.ru
2. Артикулы в формате `12345/75` будут автоматически подсвечены
3. При наведении на артикул появится всплывающая подсказка с информацией:
   - Остаток на сегодня
   - Количество заказов
   - Выкуплено/Отменено
   - Процент выкупа

## API Endpoints

Расширение использует следующие API:

- `GET /api/extension/articles` - список всех артикулов из базы
- `GET /api/extension/product-info?article=12345&size=75` - информация о товаре

## Требования

- Chrome браузер
- Доступ к серверу MarketPlacer (192.168.0.44:5000) из локальной сети
